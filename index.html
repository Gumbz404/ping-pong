Rules of the game:

Ping Pong is set to work based on the following rules:

      When the user enters a value divisible by 3, the page displays "ping"
      When the user enters a value divisible by 5, the page displays "pong"
      When the user enters a value that is divisible by 15, the page displays "pingpong"
